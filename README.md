Clash of Bots
=========

**What is Clash of Bots ?**

- Clash of Bots is a open-source and free bot for Clash of Clans.

**Which programming language is used ?**

- It's coded in C# with the .Net framework (4.5) on Visual Studio 2013 Ultimate.
